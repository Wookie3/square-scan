# Retail-Sync Pro

Retail-Sync Pro is a high-performance, mobile-first inventory management and price comparison tool. It allows warehouse staff to scan barcodes using their device's camera to instantly compare internal stock levels with live market pricing (via web scraping).

## Features

- **Mobile Barcode Scanner**: Integrated WebRTC camera scanning with haptic feedback.
- **Dual-Data Fetching**: Parallel loaders fetch internal Supabase data and live web data.
- **Live Scraper**: Server-side Playwright engine for real-time price checks on Mark's.
- **Industrial UI**: Clean, high-contrast interface designed for warehouse environments.

## Tech Stack

- **Frontend**: TanStack Start (React) + Tailwind CSS 4.0
- **Backend**: TanStack Server Functions (RPCs)
- **Database**: Supabase (PostgreSQL)
- **Browser Automation**: Playwright (Headless)
- **Scanning**: html5-qrcode

## Getting Started

### 1. Prerequisites
- Node.js (v18+)
- A Supabase Project

### 2. Environment Setup
Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. Database Setup (Supabase)
Run the following SQL in your Supabase SQL Editor to create the required table:

```sql
CREATE TABLE "Inventory" (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  "Style Number" TEXT,
  "Style" TEXT,
  "Colour" TEXT,
  "Size" TEXT,
  "Size 2" TEXT,
  "UPC" TEXT,
  "Category" TEXT,
  "SKU" TEXT,
  "Category Number" TEXT,
  "SKU Status" TEXT,
  "In Catalog" BOOLEAN,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Enable Row Level Security
ALTER TABLE "Inventory" ENABLE ROW LEVEL SECURITY;

-- Create policy for public read access
CREATE POLICY "Allow public read access" 
ON "Inventory" 
FOR SELECT 
USING (true);

-- Insert dummy data for testing
INSERT INTO "Inventory" ("Style Number", "Style", "Colour", "Size", "Size 2", "UPC", "Category", "SKU", "Category Number", "SKU Status", "In Catalog")
VALUES ('77561', 'HH OXFORD UNLINED BIB OVERALLS', 'Black', '32', '30', '7040056309347', 'WORKWEAR', '410027547516', '1A', 'Active', true);
```

### 4. Installation & Development

```bash
# Install dependencies
npm install

# Install Playwright browser
npx playwright install chromium

# Start development server
npm run dev
```
